# 공강메이트 - 대학생 공강 활용 추천 앱

## 프로젝트 소개

공강메이트는 대학생의 수업 시간표를 기반으로 공강 시간을 자동 분석하고, 공강 길이와 사용자의 관심 활동 정보를 반영하여 적합한 활동을 추천하는 모바일 앱입니다.

사용자는 학교와 관심 활동 태그를 설정한 뒤 학기별 시간표를 등록할 수 있으며, 앱은 수업 사이의 빈 시간을 계산해 공강을 분류하고 추천 활동을 제공합니다.

본 프로젝트는 개인 프로젝트로 진행했으며, 기획부터 기능 흐름 설계, 백엔드 API 구현, 모바일 화면 연동까지 전체 서비스 구조를 직접 경험하는 것을 목표로 했습니다.

## 주요 기능

### 1. 학교 및 관심 태그 설정

- 앱 최초 실행 시 학교 선택
- 사용자의 관심 활동 태그 설정
- 추천 기능에서 관심 태그를 반영할 수 있도록 사용자 입력 정보 관리

### 2. 시간표 등록 및 관리

- 학기별 과목 정보 입력
- 요일, 시작 시간, 종료 시간, 강의실 단위로 수업 정보 관리
- 과목 등록, 수정, 삭제 흐름 구현

### 3. 공강 시간 자동 계산

- 사용자가 입력한 요일별 수업 시간을 기준으로 수업 사이의 빈 시간 탐색
- 월요일부터 금요일까지의 시간표를 기준으로 공강 시간 계산
- 수업 시작 시간과 종료 시간을 비교해 공강 구간 도출

### 4. 공강 분류

계산된 공강 시간을 길이에 따라 분류했습니다.

- 짧은 공강
- 유효 공강
- 여유 공강
- 자유 시간

공강 길이에 따라 추천 가능한 활동의 범위를 다르게 판단할 수 있도록 구성했습니다.

### 5. 활동 추천

- 계산된 공강 시간 반영
- 활동별 최소 소요 시간 반영
- 사용자의 관심 태그 반영
- 사전에 정의한 날씨 조건 반영
- 추천 점수를 계산해 우선순위에 따라 활동 카드 제공

### 6. 주요 화면

- 홈 화면: 오늘의 공강 요약 제공
- 시간표 화면: 주간 시간표 및 공강 블록 표시
- 추천 화면: 공강별 활동 카드와 카테고리 필터 제공

## 추천 로직 개요

공강메이트의 추천 기능은 단순히 활동 목록을 보여주는 방식이 아니라, 사용자의 시간표와 관심 정보를 함께 반영하도록 설계했습니다.

### 1. 공강 계산

사용자가 입력한 수업 정보를 요일별로 정렬한 뒤, 각 수업의 종료 시간과 다음 수업의 시작 시간을 비교해 빈 시간을 계산합니다.

### 2. 공강 분류

계산된 공강 시간을 길이에 따라 분류하고, 각 공강에 적합한 활동 추천이 가능하도록 기준을 나눴습니다.

### 3. 추천 점수 계산

추천 활동은 다음 요소를 반영해 우선순위를 정했습니다.

- 공강 시간
- 활동별 최소 소요 시간
- 사용자 관심 태그
- 사전에 정의한 날씨 조건

이를 통해 같은 공강 시간이라도 사용자 관심사에 따라 다른 추천 결과가 제공되도록 구성했습니다.

## 개발 과정에서 중점적으로 고려한 부분

### 시간표 데이터를 활용한 기능 구현

단순히 사용자가 입력한 시간표를 저장하는 것에서 끝나지 않고, 시간표 데이터를 바탕으로 공강 시간을 계산하고 추천 기능으로 연결하는 흐름을 구현하고자 했습니다.

### 사용자 데이터 기반 맞춤형 추천

공강 길이와 관심 태그를 함께 반영해 사용자마다 다른 추천 결과를 제공할 수 있도록 구성했습니다. 이를 통해 단순 CRUD 기능을 넘어 사용자 데이터 기반 맞춤형 기능 구현을 경험했습니다.

### 새로운 기술 환경 학습

기존에는 Spring Boot 중심으로 백엔드 프로젝트를 진행했지만, 다양한 실무 기술 환경에 대비하기 위해 Node.js와 Express를 활용해 백엔드 API를 구현했습니다. 또한 React Native와 연동하며 모바일 앱의 데이터 흐름을 직접 확인했습니다.
